# TopDownVerse

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

## How to contribute

Edit sections in the `/docs` directory and create a pull request describing your changes.

Use Issues to discuss organization, potential figures, and discussion of papers.

Focus Areas of this tutorial include topics relevant to top-down mass spectrometry:

1. Theory of top-down mass spectrometry
2. Sample preparation
3. Protein separations
4. Mass spectrometry instrumentation
5. Acquisition
6. Mass deconvolution
7. Search and false discovery control
8. Quantification
9. Native vs denatured
10. Imaging MS
11. Common application areas
      - Antibodies
      - Histone
      - Ribosomes
      - Viral particles
